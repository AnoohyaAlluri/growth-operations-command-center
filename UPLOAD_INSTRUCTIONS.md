# Upload Instructions

Upload the folders and files in this starter package to the root of the GitHub repository.

Do not replace the repository's existing visual README.

Recommended commit message:

`Add synthetic Command Center analytics pipeline`

After upload, the repository root should include:

- data/
- src/
- dashboard/
- docs/
- images/
- tests/
- outputs/
- run_pipeline.py
- requirements.txt
- .gitignore
- LICENSE
